## How To Install

1. **Clone Repository**
```bash
git clone https://github.com/ariefafwan/digitallibrary.git
composer install
npm install
copy .env.example .env
```

3. **Instalasi**
```git bash
php artisan key:generate
php artisan migrate
php artisan db:seed
```

4. **Serve**
```command
php artisan serve
```
